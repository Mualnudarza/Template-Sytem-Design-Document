# 3. Design Views

Tuliskan detail teknis arsitektur yang cukup untuk diimplementasikan oleh pengembang. Gunakan format template di bawah ini untuk setiap elemen desain. Pastikan setiap elemen memiliki ID unik dan referensikan ID persyaratan SRS jika tersedia.

- **ID**: [NNN]-{Judul}
- **Title**: Nama singkat dan deskriptif dari view tersebut.
- **Viewpoint**: Sebutkan jenis viewpoint-nya (merujuk pada bab 2.2).
- **Representation**: Masukkan representasi desain berupa deskripsi teks, diagram, atau kombinasi keduanya.
- **More Information**: Tambahkan konteks tambahan atau link ke artefak terkait.
